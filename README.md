# Portfolio Website

This is my personal portfolio built with [Next.js](https://nextjs.org).  
It showcases my skills, projects, and experience as a developer.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Features

- Responsive modern UI
- Smooth animations and transitions
- Section-based navigation (About, Skills, Projects, Services, Contact)
- Mobile-friendly design
- Smooth scroll experience
- Interactive navigation with active section highlighting

## Getting Started

To run this project locally:

```bash
npm install
npm run dev

Then open:

http://localhost:3000
Project Structure
app/page.tsx – Main page
components/ – Reusable UI components
public/ – Static assets (images, resume, etc.)
Deployment
